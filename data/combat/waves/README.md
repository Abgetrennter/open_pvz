# Combat Wave Data

This directory stores reusable wave-facing combat data.

## Rules

- `waves/` is for formal wave definitions and their spawn composition.
- Wave resources should focus on pacing and spawn structure.
- Validation-only single-purpose waves should stay with their validation scenarios unless they become reusable content.
