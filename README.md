# Open PVZ - 错误技系统

基于Godot 4.x开发的植物大战僵尸同人游戏，采用随机生成的"错误技"系统。

## 项目概述

本项目实现了一个完全随机生成的植物技能系统，植物的能力由触发器和效果树动态组合而成，每次生成的植物都独一无二。

## 核心特性

- **效果一等公民**：效果可递归嵌套，通过槽位机制实现无限组合
- **触发器事件插座**：触发器只回答"何时触发"，效果回答"做什么"
- **三层生成器**：触发器抽取 → 效果树绑定 → 递归构建
- **熵灌注命名**：每株植物都有独特的乱码名称
- **事件驱动架构**：基于Godot信号系统的事件驱动执行

## 快速开始

### 环境要求

- Godot 4.x（推荐4.2或更高版本）
- Windows 10/11（或其他支持的平台）

### 运行项目

1. 下载并安装Godot 4.x：https://godotengine.org/download
2. 克隆或下载本项目
3. 用Godot打开项目目录
4. 点击"播放"按钮运行项目

### 测试植物生成

1. 运行项目后，点击"生成植物"按钮
2. 查看生成的植物信息，包括：
   - 植物名称（熵灌注生成的乱码）
   - 稀有度（常见/稀有/史诗/传说）
   - 触发器列表
   - 效果树结构

## 核心系统说明

### 触发器系统

触发器定义"何时执行效果"，支持以下类型：

- `periodically`：周期性触发（最常见）
- `when_damaged`：受击触发（较常见）
- `on_death`：死亡触发（稀有）
- `on_click`：点击触发（最稀有，手动）

### 效果系统

效果定义"做什么"，支持以下类型：

- `shoot`：发射子弹
- `damage`：直接伤害
- `explode`：范围爆炸
- `summon`：召唤实体

效果可以通过槽位递归嵌套，形成效果树。

### 命名系统

植物名称采用熵灌注命名法：

- **前缀**：§（稀有）/¶（史诗）/†（传说）/空（常见）
- **主体**：7字符随机字符串
- **分隔符**：- / . / _
- **后缀**：α（周期性）/Ω（死亡）/β（受击）/δ（点击）

示例：`§kQ3.9xFα` 表示一个稀有植物，周期性触发。
