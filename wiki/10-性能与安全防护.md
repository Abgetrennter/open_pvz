## 硬性边界

### 1. 深度上限 = 3

**规则**

任何效果树递归深度超过3，自动在效果槽位填 `null`。

**实现**

```csharp
EffectNode BuildEffectTree(EffectDef def, int depth) {
    var node = new EffectNode { /* ... */ };

    // 深度限制
    if (depth >= 3) {
        return node;
    }

    // 填充槽位
    foreach (var slot in def.slots) {
        if (slot.type == "effect") {
            var childEffectId = SelectChildEffect(slot.allowed_types);
            node.children[slot.name] = BuildEffectTree(
                EffectDefLibrary.Get(childEffectId),
                depth + 1
            );
        }
    }

    return node;
}
```

**原因**

- 防止组合爆炸
- 保证性能可预测
- 限制玩家认知负担

### 2. 事件连锁深度 = 5

**规则**

运行时连锁反应超过5层，事件静默丢弃。

**实现**

```csharp
class EffectExecutor {
    private const int MAX_CHAIN_DEPTH = 5;

    public static EffectResult Execute(EffectNode node, Context context, int depth = 0) {
        // 深度检查
        if (depth > MAX_CHAIN_DEPTH) {
            Debug.LogWarning($"Chain depth exceeded: {depth}");
            return new EffectResult {
                success = false,
                terminated = true,
                reason = "Chain depth exceeded"
            };
        }

        // 执行效果
        // ...

        // 递归执行子效果
        foreach (var child in node.children.Values) {
            var childResult = Execute(child, context, depth + 1);
            if (childResult.terminated) {
                break;
            }
        }

        return result;
    }
}
```

**原因**

- 防止无限循环
- 避免性能爆炸
- 玩家感知为"能量耗尽"

### 3. ManualTrigger互斥

**规则**

若生成时绑定两个手动触发器，第二个被静默丢弃。

**实现**

```csharp
bool CheckTriggerMutualExclusion(List<TriggerInstance> existing, TriggerDef newDef) {
    // 检查手动触发器互斥
    if (newDef.tags.Contains("manual")) {
        if (existing.Any(t => t.def.tags.Contains("manual"))) {
            Debug.LogWarning("Manual trigger conflict detected, dropping second trigger");
            return false;
        }
    }

    return true;
}
```

**原因**

- 防止手动触发冲突
- 保证用户体验一致性

### 2. 参数白名单

**规则**

槽位定义限制参数范围，防止恶意值。

**实现**

```csharp
class ParameterValidator {
    public static bool Validate(string slotName, object value, SlotDef slotDef) {
        if (slotDef.type != "value") {
            return true;
        }

        switch (slotDef.value_type) {
            case "int":
                int intValue = (int)value;
                return intValue >= slotDef.min && intValue <= slotDef.max;
            case "float":
                float floatValue = (float)value;
                return floatValue >= (float)slotDef.min && floatValue <= (float)slotDef.max;
            case "bool":
                return value is bool;
            case "string":
                return value is string;
            default:
                return false;
        }
    }
}
```

**示例**

```json
{
  "name": "damage",
  "type": "value",
  "value_type": "int",
  "min": 0,
  "max": 999
}
```

**防护**

- 防止过大数值
- 防止类型错误
- 防止恶意注入

### 3. 事件白名单

**规则**

事件名由核心层广播，外部只能订阅。

**实现**

```csharp
class EventManager {
    private static HashSet<string> _allowedEvents = new HashSet<string> {
        "plant.planted",
        "plant.died",
        "plant.damaged",
        "projectile.hit",
        "zombie.died",
        "game.tick"
        // ...

    public static bool IsEventAllowed(string eventName) {
        return _allowedEvents.Contains(eventName);
    }

    public static void Broadcast(string eventName, EventData eventData) {
        if (!IsEventAllowed(eventName)) {
            Debug.LogError($"Event not allowed: {eventName}");
            return;
        }

        // 广播事件
        // ...
    }
}
```

**防护**

- 防止未授权事件
- 防止事件注入

### 4. 沙盒执行

**规则**

策略接口无文件/网络权限，纯计算。

**实现**

class EffectStrategy {
    // 纯计算接口，无副作用

## 性能监控

### 性能分析器

```csharp
class PerformanceProfiler {
    private static Dictionary<string, float> _executionTimes = new();
    private static Dictionary<string, int> _executionCounts = new();

    public static void Profile(string name, Action action) {
        var stopwatch = Stopwatch.StartNew();
        action();
        stopwatch.Stop();

        if (!_executionTimes.ContainsKey(name)) {
            _executionTimes[name] = 0;
            _executionCounts[name] = 0;
        }

        _executionTimes[name] += stopwatch.ElapsedMilliseconds;
        _executionCounts[name]++;
    }

    public static void PrintReport() {
        Debug.Log("=== Performance Report ===");
        foreach (var pair in _executionTimes.OrderByDescending(p => p.Value)) {
            float avgTime = pair.Value / _executionCounts[pair.Key];
            Debug.Log($"{pair.Key}: {pair.Value}ms total, {avgTime}ms avg, {_executionCounts[pair.Key]} calls");
        }
    }
}
```

### 使用示例

```csharp
PerformanceProfiler.Profile("ExecuteEffectTree", () => {
    EffectExecutor.Execute(effectTree, context);
});
```

## 内存管理

### 对象池

```csharp
class ObjectPool<T> where T : new() {
    private Stack<T> _pool = new Stack<T>();

    public T Get() {
        if (_pool.Count > 0) {
            return _pool.Pop();
        }
        return new T();
    }

    public void Return(T obj) {
        _pool.Push(obj);
    }
}

// 使用
var effectNodePool = new ObjectPool<EffectNode>();
var node = effectNodePool.Get();
// 使用node
effectNodePool.Return(node);
```
