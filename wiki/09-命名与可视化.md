# 命名与可视化

> 熵灌注命名法与UI展示

---

## 概述

命名与可视化系统负责为随机生成的植物创建独特的名称和图标，通过"熵灌注命名法"生成可复现的乱码名称，并通过UI元素传达植物的稀有度和触发方式。

---

## 熵灌注命名法

### 命名规则

```plaintext
输入：效果树遍历串 + 时间戳
输出：§kQ3.9xFα  （11字符乱码）
  - Prefix §/¶/†：稀有效果数量决定
  - Body 7字符：随机62进制
  - Suffix α/Ω/β：触发器类型
```

### 命名结构

| 部分 | 长度 | 说明 | 示例 |
|------|------|------|------|
| Prefix | 1字符 | 稀有度标识 | §, ¶, † |
| Body | 7字符 | 随机字符 | kQ3.9xF |
| Suffix | 1字符 | 触发器类型 | α, Ω, β |

---

### 前缀（Prefix）—— 稀有度标识

| 前缀 | 稀有度 | 说明 |
|------|--------|------|
| 无 | 普通 | 无稀有效果 |
| § | 稀有 | 含1-2个稀有效果 |
| ¶ | 史诗 | 含3-4个稀有效果 |
| † | 传说 | 含5+个稀有效果 |

### 前缀生成规则

```csharp
string GeneratePrefix(EffectNode rootNode) {
    int rareCount = CountRareEffects(rootNode);

    if (rareCount >= 5) return "†";
    if (rareCount >= 3) return "¶";
    if (rareCount >= 1) return "§";
    return "";
}

int CountRareEffects(EffectNode node) {
    int count = EffectDefLibrary.Get(node.effect_id).tags.Contains("rare") ? 1 : 0;
    foreach (var child in node.children.Values) {
        if (child.effect_id != "null") {
            count += CountRareEffects(child);
        }
    }
    return count;
}
```

---

### 主体（Body）—— 随机字符

**字符集**

```
0-9: 10个
a-z: 26个
A-Z: 26个
.: 1个
总计: 62个字符
```

**生成规则**

```csharp
string GenerateBody(string traversalString, long timestamp) {
    var hash = Hash(traversalString + timestamp);
    var body = new StringBuilder();

    for (int i = 0; i < 7; i++) {
        int index = (hash >> (i * 8)) % 62;
        body.Append(CHARSET[index]);
    }

    return body.ToString();
}

private static readonly string CHARSET = "0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ.";
```

**特性**

- 相同输入 → 相同输出，可复现
- 7字符长度，足够唯一性
- 62进制，包含数字、字母、点号

---

### 后缀（Suffix）—— 触发器类型

| 后缀 | 触发器类型 | 说明 |
|------|-----------|------|
| α | 周期性 | periodically |
| Ω | 死亡 | on_death |
| β | 受伤 | when_damaged |
| γ | 点击 | on_click |
| δ | 手动 | manual |

**生成规则**

```csharp
string GenerateSuffix(TriggerInstance primaryTrigger) {
    switch (primaryTrigger.def_id) {
        case "periodically": return "α";
        case "on_death": return "Ω";
        case "when_damaged": return "β";
        case "on_click": return "γ";
        case "manual": return "δ";
        default: return "?";
    }
}
```

## Tooltip动态解码

### Tooltip结构

```
§kQ3.9xFα
━━━━━━━━━
稀有度: §（含稀有效果）
触发: α（周期性，每3秒）
结构: 发射 → 爆炸 → 召唤
深度: 3层
```

### 解码实现

```csharp
class TooltipGenerator {
    public static string GenerateTooltip(string name, EffectNode rootNode, TriggerInstance trigger) {
        var tooltip = new StringBuilder();
        tooltip.AppendLine(name);
        tooltip.AppendLine("━━━━━━━━━");

        // 解析前缀
        var prefix = name[0].ToString();
        tooltip.AppendLine($"稀有度: {GetRarityDescription(prefix)}");

        // 解析后缀
        var suffix = name[name.Length - 1].ToString();
        tooltip.AppendLine($"触发: {GetTriggerDescription(suffix, trigger)}");

        // 解析效果树
        tooltip.AppendLine($"结构: {GetEffectTreeDescription(rootNode)}");

        // 计算深度
        tooltip.AppendLine($"深度: {GetTreeDepth(rootNode)}层");

        return tooltip.ToString();
    }

    private static string GetRarityDescription(string prefix) {
        switch (prefix) {
            case "§": return "§（含1-2个稀有效果）";
            case "¶": return "¶（含3-4个稀有效果）";
            case "†": return "†（含5+个稀有效果）";
            default: return "普通（无稀有效果）";
        }
    }

    private static string GetTriggerDescription(string suffix, TriggerInstance trigger) {
        var interval = trigger.condition_values.ContainsKey("interval")
            ? trigger.condition_values["interval"].ToString()
            : "未知";

        switch (suffix) {
            case "α": return $"α（周期性，每{interval}秒）";
            case "Ω": return "Ω（死亡时）";
            case "β": return "β（受伤时）";
            case "γ": return "γ（点击时）";
            case "δ": return "δ（手动触发）";
            default: return "未知";
        }
    }

    private static string GetEffectTreeDescription(EffectNode node) {
        var effects = new List<string>();
        CollectEffects(node, effects);
        return string.Join(" → ", effects);
    }

    private static void CollectEffects(EffectNode node, List<string> effects) {
        if (node.effect_id != "null") {
            effects.Add(node.effect_id);
        }
        foreach (var child in node.children.Values) {
            CollectEffects(child, effects);
        }
    }

    private static int GetTreeDepth(EffectNode node) {
        if (node.effect_id == "null") return 0;
        int maxDepth = 1;
        foreach (var child in node.children.Values) {
            maxDepth = Mathf.Max(maxDepth, 1 + GetTreeDepth(child));
        }
        return maxDepth;
    }
}
```

## 命名复现

### 复现原理

由于命名是基于效果树遍历串和时间戳的哈希，因此相同的输入会产生相同的输出。

### 复现流程

```plaintext
1. 序列化效果树为遍历串
2. 获取生成时间戳
3. 计算哈希
4. 生成名称
```

### 序列化实现

```csharp
class EffectTreeSerializer {
    public static string Serialize(EffectNode node) {
        var sb = new StringBuilder();
        SerializeNode(node, sb);
        return sb.ToString();
    }

    private static void SerializeNode(EffectNode node, StringBuilder sb) {
        sb.Append(node.effect_id);
        sb.Append("(");

        // 序列化参数
        sb.Append("params:");
        foreach (var pair in node.params) {
            sb.Append($"{pair.Key}={pair.Value},");
        }

        // 序列化子节点
        sb.Append("children:");
        foreach (var pair in node.children) {
            sb.Append($"{pair.Key}=");
            SerializeNode(pair.Value, sb);
            sb.Append(",");
        }

        sb.Append(")");
    }
}
```

## 分享码

### 分享码格式

```
分享码 = Base64(遍历串 + 时间戳)
```

### 生成分享码

```csharp
class ShareCodeGenerator {
    public static string Generate(EffectNode rootNode, long timestamp) {
        var traversalString = EffectTreeSerializer.Serialize(rootNode);
        var data = $"{traversalString}|{timestamp}";
        var bytes = Encoding.UTF8.GetBytes(data);
        return Convert.ToBase64String(bytes);
    }
}
```

### 解析分享码

```csharp
class ShareCodeParser {
    public static (EffectNode rootNode, long timestamp) Parse(string shareCode) {
        var bytes = Convert.FromBase64String(shareCode);
        var data = Encoding.UTF8.GetString(bytes);
        var parts = data.Split('|');

        var rootNode = EffectTreeDeserializer.Deserialize(parts[0]);
        var timestamp = long.Parse(parts[1]);

        return (rootNode, timestamp);
    }
}
```
