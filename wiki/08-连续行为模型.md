# 连续行为模型

> 叠加式动力学详解

---

## 概述

连续行为模型采用"贡献项叠加"的方式，而不是单一轨迹模式切换。每帧收集所有组件输出的速度增量或力，合成后更新velocity和position。

---

## 叠加式动力学

### 核心思想

推荐使用"贡献项叠加"的方式，而不是单一轨迹模式切换。

### 每帧流程

```plaintext
1. 收集所有组件输出的速度增量或力
2. 合成
3. 更新velocity
4. 更新position
```

### 数学表达

```
v(t+1) = v(t) + ΣΔv_i
position += velocity * dt
```

---

## 连续行为的原则

### 原则1：组件只能输出贡献项

组件不能直接破坏全局状态，只能输出贡献项。

**错误示例**

```csharp
// 错误：直接修改全局速度
class StraightTrajectory {
    public void Update(Projectile projectile) {
        projectile.velocity = new Vector3(1, 0, 0) * speed; // 直接修改
    }
}
```

**正确示例**

```csharp
// 正确：输出速度增量
class StraightTrajectory {
    public Vector3 GetVelocityDelta(Projectile projectile) {
        return new Vector3(1, 0, 0) * speed;
    }
}
```

---

### 原则2：所有贡献项应共享同一时间基准

所有贡献项应基于相同的时间基准（dt），避免时间不一致。

**示例**

```csharp
class Projectile {
    private List<TrajectoryComponent> _components = new();

    public void Update(float dt) {
        // 收集所有贡献项
        Vector3 totalDelta = Vector3.zero;
        foreach (var component in _components) {
            totalDelta += component.GetVelocityDelta(this, dt); // 传入dt
        }

        // 合成
        velocity += totalDelta;
        position += velocity * dt;
    }
}
```

---

### 原则3：叠加顺序尽量可交换

贡献项的叠加顺序应尽量可交换，减少顺序耦合。

**示例**

```csharp
// 可交换：线性叠加
velocity += straightTrajectory.GetDelta();
velocity += magneticTrajectory.GetDelta();
velocity += sineTrajectory.GetDelta();

// 不可交换：依赖顺序（尽量避免）
velocity = straightTrajectory.Apply(velocity);
velocity = magneticTrajectory.Apply(velocity);
velocity = sineTrajectory.Apply(velocity);
```

## 轨迹实现方式

### 推荐方式

推荐把"直线、正弦、磁力、追踪"等都视为力或速度贡献，而不是独立轨迹类型。

### 优势

| 优势 | 说明 |
|------|------|
| 轨迹可叠加 | 多种轨迹可以同时生效 |
| 轨迹可组合 | 可以自由组合不同的轨迹效果 |
| 轨迹可导出为数据 | 轨迹参数可以序列化为JSON |

---

## 轨迹组件示例

### 直线轨迹

```csharp
class StraightTrajectory : TrajectoryComponent {
    public float speed;

    public Vector3 GetVelocityDelta(Projectile projectile, float dt) {
        return projectile.direction * speed * dt;
    }
}
```

---

### 正弦轨迹

```csharp
class SineTrajectory : TrajectoryComponent {
    public float amplitude;
    public float frequency;
    private float _time;

    public Vector3 GetVelocityDelta(Projectile projectile, float dt) {
        _time += dt;
        float offset = Mathf.Sin(_time * frequency) * amplitude;
        return projectile.direction.Perpendicular() * offset * dt;
    }
}
```

---

### 磁力轨迹

```csharp
class MagneticTrajectory : TrajectoryComponent {
    public float strength;
    public float range;

    public Vector3 GetVelocityDelta(Projectile projectile, float dt) {
        var targets = GetTargetsInRange(projectile.position, range);
        if (targets.Count == 0) return Vector3.zero;

        var nearest = targets.OrderBy(t => Vector3.Distance(t.position, projectile.position)).First();
        var direction = (nearest.position - projectile.position).normalized;
        return direction * strength * dt;
    }
}
```

---

### 追踪轨迹

```csharp
class HomingTrajectory : TrajectoryComponent {
    public float turnRate;
    public Entity target;

    public Vector3 GetVelocityDelta(Projectile projectile, float dt) {
        if (target == null) return Vector3.zero;

        var desiredDirection = (target.position - projectile.position).normalized;
        var currentDirection = projectile.direction;
        var newDirection = Vector3.RotateTowards(
            currentDirection,
            desiredDirection,
            turnRate * dt,
            0
        );

        return (newDirection - currentDirection) * projectile.speed * dt;
    }
}
```

### 动态加载

```csharp
class TrajectoryFactory {
    public static List<TrajectoryComponent> CreateFromConfig(JSONObject config) {
        var components = new List<TrajectoryComponent>();

        foreach (var componentConfig in config["components"].AsArray) {
            var type = componentConfig["type"].AsString;
            var component = CreateComponent(type, componentConfig);
            components.Add(component);
        }

        return components;
    }

    private static TrajectoryComponent CreateComponent(string type, JSONObject config) {
        switch (type) {
            case "straight":
                return new StraightTrajectory {
                    speed = config["speed"].AsFloat
                };
            case "sine":
                return new SineTrajectory {
                    amplitude = config["amplitude"].AsFloat,
                    frequency = config["frequency"].AsFloat
                };
            case "magnetic":
                return new MagneticTrajectory {
                    strength = config["strength"].AsFloat,
                    range = config["range"].AsFloat
                };
            case "homing":
                return new HomingTrajectory {
                    turnRate = config["turnRate"].AsFloat,
                    target = GetTarget(config["targetId"].AsString)
                };
            default:
                throw new ArgumentException($"Unknown trajectory type: {type}");
        }
    }
}
```

## 轨迹注册

### 动态注册

```csharp
class TrajectoryRegistry {
    private static Dictionary<string, Func<JSONObject, TrajectoryComponent>> _creators = new();

    public static void Register(string type, Func<JSONObject, TrajectoryComponent> creator) {
        _creators[type] = creator;
    }

    public static TrajectoryComponent Create(string type, JSONObject config) {
        if (!_creators.ContainsKey(type)) {
            throw new ArgumentException($"Unknown trajectory type: {type}");
        }
        return _creators[type](config);
    }
}
```

### MOD注册

```csharp
// MOD初始化时注册自定义轨迹

    });
}
```

## 轨迹调试

### 可视化

```csharp
class TrajectoryDebugger {
    public static void DrawTrajectory(Projectile projectile, float duration = 1f) {
        var points = SimulateTrajectory(projectile, duration);

        for (int i = 0; i < points.Count - 1; i++) {
            Debug.DrawLine(points[i], points[i + 1], Color.yellow);
        }
    }

    private static List<Vector3> SimulateTrajectory(Projectile projectile, float duration) {
        var points = new List<Vector3>();
        var simulated = projectile.Clone();

        for (float t = 0; t < duration; t += 0.1f) {
            points.Add(simulated.position);
            simulated.Update(0.1f);
        }

        return points;
    }
}
```

### 性能分析

```csharp
class TrajectoryProfiler {
    private static Dictionary<string, float> _executionTimes = new();

    public static void Profile(TrajectoryComponent component, Action action) {
        var stopwatch = Stopwatch.StartNew();
        action();
        stopwatch.Stop();

        var typeName = component.GetType().Name;
        if (!_executionTimes.ContainsKey(typeName)) {
            _executionTimes[typeName] = 0;
        }
        _executionTimes[typeName] += stopwatch.ElapsedMilliseconds;
    }

    public static void PrintReport() {
        foreach (var pair in _executionTimes.OrderByDescending(p => p.Value)) {
            Debug.Log($"{pair.Key}: {pair.Value}ms");
        }
    }
}
```
