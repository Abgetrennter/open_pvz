# 第六阶段-正式交互矩阵-v1

> 这份文档用于承接第六阶段的 `Epic C`。它回答的不是“当前有哪些单位”，而是“这些单位之间为什么会形成正式对抗关系，以及当前第一轮正式交互矩阵已经成立到什么程度”。

---

## 文档定位

这篇文档主要回答：

- 当前正式植物与僵尸大类如何发生对抗
- 哪些关系属于克制，哪些属于压制，哪些属于交换
- 当前第一轮正式交互矩阵已经成立了什么
- 哪些关系仍属于下一轮缺口

这篇文档不负责：

- 代替数值平衡表
- 代替完整关卡设计说明

---

## 一句话原则

第六阶段开始后，单位不能只按“会不会攻击”来定义，而应按：

> 它在正式战斗里制造什么压力，或提供什么回应。

## 当前矩阵摘要

| 植物大类 | 主要回应对象 | 当前关系判断 |
|------|------|------|
| 基础输出型 | 基础推进型 | 基础交换 |
| 高频输出型 | 基础推进型 / 快速压迫型 | 弱克制 |
| 抛投覆盖型 | 护甲型 / 后排威胁 | 条件克制 |
| 资源生产型 | 长局与阵地建设 | 间接支撑 |
| 防御 / 拦截型 | 基础推进型 / 特殊攻击型 | 承压回应 |
| 控制型 | 快速压迫型 | 明确克制 |
| 支援 / 基础设施型 | 水路 / 屋顶 / 空域限制 | 场地回应 |
| 召唤 / 分体型 | 中高压推进 | 条件克制 |

| 僵尸大类 | 主要压制对象 | 当前关系判断 |
|------|------|------|
| 基础推进型 | 低火力 / 无防线 | 基础压力 |
| 快速压迫型 | 慢节奏布阵 | 明确压制 |
| 高血 / 护甲型 | 单点低压输出 | 明确压制 |
| 特殊攻击型 | 单一守线节奏 | 明确压制 |
| 空中型 | 缺乏对空能力的阵地 | 明确压制 |
| 支援 / 干扰型 | 稳定输出与固定节奏 | 明确压制 |

## 下一轮缺口

- 真正光环型支援植物仍未进入正式矩阵
- 更强的支援 / 召唤型僵尸仍未系统展开
- “范围清场对召唤海”的正式关系仍需更直接验证
- 更复杂的高度压制关系仍可继续补位

---

## 一句话总结

第六阶段当前第一轮正式交互矩阵已经不再只是“谁伤害高”，而是已经开始形成：

> 控制对快压、召唤对重压、特殊攻击对守线、干扰对稳定输出、空中对覆盖缺口的正式对抗结构。
